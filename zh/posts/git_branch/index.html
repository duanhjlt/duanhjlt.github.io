git 分支操作常用命令

2024-01-04 · 1 分钟 · duanhongjin | Suggest Changes
目录
  • 查看所有分支
  • 查看远程分支
  • 查看本地分支所关联的远程分支
  • 重命名本地分支
  • 删除远程分支
  • 推送新的分支
  • 设置关联

查看所有分支#

$ git branch -a

查看远程分支#

$ git branch -r

查看本地分支所关联的远程分支#

$ git branch -vv

重命名本地分支#

$ git branch -m old_branch new_branch

删除远程分支#

$ git branch -d -r origin/old_branch
$ git push origin :old_branch

推送新的分支#

$ git push -u origin new_branch

设置关联#

$ git branch --set-upstream-to=origin/branch branch
